# Directory Overview

Holds the video data of all the games. There should be video data gathered by web scraping and then those videos to be split into random segments to be able to create our training data

### Video Overview

The games should be from anything from the past 25 years and should be any form of quality soccer from any top leagues and competitions such as the top 5 European leagues as well as prolific international competitions in addition to other tournaments such as the champions league and domestic competitions
